Making a big entrance...

Normally, when an element is rendered, it just sort of gets plonked on the page. With Ractive you have more control: you can specify intro transitions:

<div intro='fade'>This div will fade into view</div>

Try adding intros to the three buttons in the template, choosing from fade, slide and fly.

As with custom events, transitions are distributed as plugins, which you can download here to use in your project. You can create your own transitions – see this wiki page for more info.

Caution! Some of the built-in transitions currently use CSS animations, which don't work in IE9 and below, and which don't look quite as slick in IE10 as in modern browsers. That's because IE is crap (yes, even IE10). Specifically, it can't animate certain styles (such as left) using CSS – the solution is to use good old interval-based animation rather than CSS transitions in these cases. Hopefully we'll be able to address this in future versions of Ractive.js. In the meantime, friends don't let friends use IE.
